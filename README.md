# AI-Powered Sentiment Analysis using Machine Learning

## Overview

This project performs Sentiment Analysis on movie reviews using Natural Language Processing (NLP) and Machine Learning techniques. The model classifies reviews as **Positive** or **Negative** based on the text content.

## Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Text Vectorization using CountVectorizer
* Sentiment Classification using Multinomial Naive Bayes
* Accuracy Evaluation and Performance Metrics
* Custom Review Prediction

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Dataset

This project uses the IMDb Movie Reviews Dataset.

Due to GitHub file size limitations, the dataset is not included in this repository.

Download the dataset from:
https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Extraction using CountVectorizer
5. Model Training
6. Prediction and Testing
7. Performance Evaluation

## Results

The model achieved approximately 85% accuracy on the test dataset and successfully classified movie reviews into positive and negative sentiments.
